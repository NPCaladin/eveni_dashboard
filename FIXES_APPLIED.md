# 수정 완료 사항

## 1. 컨설턴트 리소스 엑셀 업로드 기능 추가

### 변경 내역:
- ✅ 새 컴포넌트 생성: `components/education/consultant-resource-upload.tsx`
- ✅ 어드민 페이지에 엑셀 업로드 컴포넌트 추가
- ✅ 수동 입력 그리드(ConsultantResourceGrid)와 엑셀 업로드 모두 사용 가능

### 사용 방법:
1. 어드민 페이지 > 교육사업본부
2. "컨설턴트 리소스 엑셀 업로드" 섹션에서 파일 선택
3. 엑셀 형식: 직군, 상태(가능/불가/조율/전체마감), 비고
